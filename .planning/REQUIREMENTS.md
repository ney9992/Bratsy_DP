# Requirements — Bratsy_DP (Цифровой завод)

## v1 Requirements

### Интеграция (INT)

- [ ] **INT-01**: Пользователь может запустить симуляцию Tecnomatix Plant Simulation через кнопку в UI
- [ ] **INT-02**: Приложение получает результаты симуляции (загрузка линии, пропускная способность, время цикла) после завершения
- [ ] **INT-03**: Путь к файлу модели Plant Simulation (.spp) задаётся через панель настроек

### Пайплайн (PIPE)

- [ ] **PIPE-01**: Пользователь видит статус каждого этапа в реальном времени (Ожидает / Запущен / Завершён / Ошибка)
- [ ] **PIPE-02**: Пользователь может запустить любой отдельный этап пайплайна
- [ ] **PIPE-03**: Пользователь может остановить выполняющийся этап
- [ ] **PIPE-04**: Результаты симуляции Plant Simulation отображаются в UI после завершения этапа

### UI (UI)

- [ ] **UI-01**: Пользователь получает уведомление (в UI) при завершении этапа или возникновении ошибки
- [ ] **UI-02**: Пользователь открывает панель настроек и задаёт пути к файлам и параметры систем
- [ ] **UI-03**: Пользователь видит лог текущего выполнения этапа в реальном времени (построчно)

## v2 Requirements (деферировано)

- Vault PDM интеграция — извлечение данных изделий (Изделие, Состав, Ревизия, Чертежи)
- AutoCAD интеграция — обработка чертежей через AutoLISP-скрипты
- Excel / ТРП интеграция — Маршрут, Операции, Ресурсы, Нормы
- Запуск полного пайплайна одной кнопкой (PDM → симуляция → отчёт)
- История запусков — последние N запусков с результатами
- Anylogic интеграция — альтернатива Plant Simulation

## Out of Scope

- MES контур — не определено ПО, откладывается до следующего milestone
- BIM контур (Navisworks, Revit, Solidworks) — отдельный крупный блок, не v1
- VR / 3D-рендеры (Visual Components) — требует отдельного исследования
- LLM AI / внутренняя нейросеть проекта — долгосрочная цель
- Веб-интерфейс / мобильная версия — Windows-only среда
- Отчётная генерация (PDF/Word) — не в скоупе v1
