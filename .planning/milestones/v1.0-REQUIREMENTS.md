### Интеграция (INT)

- [x] **INT-01**: Пользователь может запустить симуляцию Tecnomatix Plant Simulation через кнопку в UI

### Пайплайн (PIPE)

- [x] **PIPE-01**: Пользователь видит статус каждого этапа в реальном времени (Ожидает / Запущен / Завершён / Ошибка)
  - *Validated in Phase 2. Outcome: pill статусы через stage-status Tauri events.*
- [x] **PIPE-02**: Пользователь может запустить любой отдельный этап пайплайна
  - *Validated in Phase 2. Outcome: run_stage Rust command + JS launch flow.*
- [x] **PIPE-03**: Пользователь может остановить выполняющийся этап
  - *Validated in Phase 2. Outcome: stop_stage + taskkill.*
- [x] **PIPE-04**: Результаты симуляции Plant Simulation отображаются в UI после завершения этапа
  - *Validated in Phase 3+4. Outcome: reportGridDyn карточки через listen('stage-results').*

### UI (UI)

  - *Validated in Phase 2. Outcome: showToast() + stage-status events.*
- [x] **UI-02**: Пользователь открывает панель настроек и задаёт пути к файлам и параметры систем
  - *Validated in Phase 1. Outcome: gear-button + sliding settings overlay + settings.json.*
- [x] **UI-03**: Пользователь видит лог текущего выполнения этапа в реальном времени (построчно)
